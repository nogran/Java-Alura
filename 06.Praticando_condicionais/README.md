## Nessa aula, aprendemos:

## Também já vimos alguns atalhos no Eclipse:

- `main + ctrl + espaço`
	- Para gerar o método `main`;
- `ctrl + shift + f`
	- para formatar o código fonte
- `sysout + ctrl + espaço`
	- para gerar a instrução `System.out.println()`.
